# Project Nadeuli

## 프로젝트 팀
- 네이버클라우드 데브옵스 양성과정 2기 Team 딸깍
- 이상민, 최태현, 김동헌, 권순재
<br/>

## 프로젝트 소개
- 프로젝트명 : 나드리
- 동네 기반 중고거래, 배달, 커뮤니티 플랫폼
<br/>

## 주요 기능
- 계정 없이 휴대폰 인증을 통한 가입 및 로그인
- 실계좌 유효성 검증 및 등록
- 동네(구) 단위 거래 및 소모임 활동 범위 설정
- 1:1 채팅
- 카카오페이, 토스 결제
- 실시간 스트리밍
- 중고, 일반 상품 배달 서비스
<br/>
